How to Play

 Sokoban is a logic puzzle game. You play a warehouse keeper (the circle). You need to put the boxes in their proper place, marked in dark-gray. However, you can only push the crates from behind, and only one crate at a time.

For more stuff, visit
https://github.com/xiaosea/Sokoban

You'll also find the latest version of this application, as well as the source code.

The actual puzzles were taken from the 'ksokoban' game in the KDE 3.0 game suite.
